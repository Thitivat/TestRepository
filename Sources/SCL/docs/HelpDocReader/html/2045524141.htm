Bndb.Kyc.DocReader Namespace

Classes

  Class Description
DocReaderException
Class DocReaderException is a custom exception class for manipulate Exception inside of DocReader. This class inherit from System.Exception class.
ImageModifier
Class ImageModifier is a class performing image such as cropping, rotation, etc. We have to use this class before do OCR to get high accuracy as much as possible. This class implements IImageModifier interface.
ImageOperationException
Class ImageOperationException is a custom exception class for throwing exception about image modifier component which consists of ImageModifier and ImageSettings classes.
ImageSettings
Class ImageSettings is a settings class for using with ImageModifier class to validates image.
OcrEngine
Class OcrEngine is a class for process image file by read the text from image then return text as result. click here for more information. This class implements IOcrEngine interface.
OcrOperationException
Class OcrOperationException is a class that inherited from DocReaderException. This class will manipulate with all exception that occur in OcrEngine.
Reader
The Reader class is component that supports another component to use the Plugin component that implemented IPluginReader. Reader component use [Bndb.Kyc.Common.Mef.PluginManager] to access/get the plugins from specific directory that will pass as a parameter on create constructor.
ReaderOperationException
The ReaderOperationException class is a custom exception class for used with Reader class This class inherit from DocReaderException class.
