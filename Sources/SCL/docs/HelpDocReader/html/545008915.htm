Bndb.Kyc.DocReader.Interfaces Namespace

Interfaces

  Interface Description
IImageModifier
Interface IImageModifier is the contract that provides the function for validate and modify the image.
IOcrEngine
Interface IPluginReader is the contract for use ocr functionality. It contains method for manipulate with image such as the function for convert to gray scale, deskew, and have process method for read the text from image and return string as result.
IReader
Interface IReader is the contract to get IPluginReader object by using MEF. The IPluginReader object will provides the functionality up to the user call GetReader with existing plugin name.
